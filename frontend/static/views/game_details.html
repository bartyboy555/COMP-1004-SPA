Game Details

Goal:

Eat pellets to grow SnakMan's tail as long as possible

Ghosts:

You must try to avoid the ghosts as they can harm Snakman's tail decreasing score

Powerups:

Eating Powerups which are purple allow you to be uneffected by ghosts and eat them for 6 seconds

Rounds:

As you eat all pellets a new round starts which reset pellets and ghosts

The more rounds you play the harder the game gets
